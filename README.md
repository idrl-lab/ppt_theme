# PPT Theme

## 介绍

考虑到 beamer 在多媒体上的短板，复刻实现了 [idrl beamer_template](https://git.idrl.site/idrl/beamer_template) 的 ppt 主题，方便大家使用.

- 标题页
- 单栏页
- 双栏页

## 用法

![guide](./guide.png)

- 下载 `idrl_ppt_theme.pptx` 并打开，按如下步骤操作

1. 视图-幻灯片母版
2. 点击左侧第 1 个母版
3. 修改作者
4. 修改日期
5. 修改标题
6. 由于 ppt 不支持自动计算总页数，请做完 ppt 之后手动修改模板中右下角的 `总页数`
